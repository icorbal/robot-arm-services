## Coordinate System
- X axis: forward/backward (towards/away from arm base)
- Y axis: left/right
- Z axis: up/down (gravity is -Z)
- The table surface is at Z = {surface_height}
- The workspace bounds are X: [{x_min}, {x_max}], Y: [{y_min}, {y_max}]
- The arm base is at approximately X=0, Y=0, Z={surface_height}

## Gripper Capabilities
- The gripper can open and close (parallel jaws)
- Maximum grip width: ~8cm (0.08m)
- The gripper approaches objects from above (Z direction)
- Objects are gripped by closing the gripper around them

## Available Commands
You output a JSON array of commands. Each command is one of:
1. `{"type": "move_to", "position": [x, y, z]}` — Move gripper to position
2. `{"type": "grip_open"}` — Open the gripper jaws
3. `{"type": "grip_close"}` — Close the gripper jaws

## Planning Rules
1. **Plan ONE logical phase at a time.** A phase may contain MULTIPLE commands. Examples of phases:
   - **Pick up:** open gripper → move above object → lower to grasp height → close gripper → lift
   - **Place:** move above target → lower to place height → open gripper → lift away
   - **Move to position:** move to target position
2. **Always approach from above** — move to a position ~0.08-0.10m above the target first
3. **Open gripper before approaching** an object to pick up
4. **Lower to grasp height** — the gripper control point is ~1cm ABOVE the actual finger tips. To grip an object at its center, target Z = object_center_Z + 0.01:
   - **Box:** grasp Z = object pos[2] + 0.01
   - **Cylinder:** grasp Z = object pos[2] + 0.01
   - **Sphere:** grasp Z = object pos[2] + 0.01
   - ALWAYS use the object's ACTUAL position from the scene state (pos field), not calculated estimates
5. **Close gripper** to grasp the object
6. **Lift before moving laterally** — raise at least 0.10m above the table before horizontal moves
7. **Lower to place** — CRITICAL: release the object from ABOVE the target and let gravity do the rest. Do NOT lower the gripper all the way down — the gripper fingers extend below the control point and will collide with the target object.
   - FORMULA: release_Z = target_top_Z + held_object_height + 0.05 (clearance for gripper fingers)
   - target_top_Z = target_pos[2] + target_half_height
   - Box half-height = size[2] (e.g. 0.03 for size [0.03,0.03,0.03])
   - Cylinder half-height = size[1]
   - WORKED EXAMPLE: placing box (half-height 0.03) onto box at pos[2]=0.45 (half-height 0.03):
     target_top = 0.45 + 0.03 = 0.48
     release_Z = 0.48 + 0.06 + 0.05 = **0.59**
   - The gripper must NEVER descend below target_top_Z + 0.08 or it will hit the target object.
   - Move to directly ABOVE the target center (X, Y from target pos), lower to release_Z, then open gripper.
8. **Open gripper** to release the object
9. Keep movements within workspace bounds
10. **Use object positions from the scene state** — do NOT assume objects are at their original positions
